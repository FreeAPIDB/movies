How it Works

The Movie API provides a list of major Bollywood movie releases for any given year. Here's how you can get the movie data:

  1. Visit the following URL format: https://freeapidb.github.io/movies/y/{year}.json
  2. Replace {year} with the desired year. For example, to get movies from 2011, use: https://freeapidb.github.io/movies/y/2011.json.
  3. The data will be returned in JSON format, which you can use for your application or analysis.
